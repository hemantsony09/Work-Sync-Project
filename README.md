# 🚀 WorkSync

WorkSync is a modern team management platform designed to simplify workforce collaboration, attendance tracking, leave management, task organization, and meeting scheduling for organizations.

It provides a centralized dashboard for employees and administrators to manage daily operations efficiently while ensuring a smooth and responsive user experience.

---

## ✨ Features

### 👤 Authentication
- Secure Login
- JWT Authentication
- Role-Based Authorization (Admin & Employee)

### 📅 Attendance Management
- Daily Punch In / Punch Out
- Attendance Tracking
- Attendance History

### 📝 Leave Management
- Apply Leave
- Leave Approval & Rejection
- Leave Status Tracking

### 📋 Task Management
- Create Tasks
- Assign Tasks
- Update Task Status
- Task Progress Tracking

### 📆 Meeting Scheduler
- Schedule Meetings
- Invite Team Members
- Meeting Notifications

### 📢 Announcement System
- Broadcast Notifications
- Company Announcements

### 👨‍💼 Admin Panel
- Employee Management
- Attendance Monitoring
- Leave Approval
- Task Monitoring
- Dashboard Analytics
